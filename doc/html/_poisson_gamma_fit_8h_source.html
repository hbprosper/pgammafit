3 // File: PoissonGammaFit.h
5 // Description:
6 //
7 // Model:
8 //
9 // d_i = Sum p_j * a_ji, i=1..M, j=1..N
10 //
11 // D_i is count corresponding to true mean d_i
12 // A_ji is count corresponding to true mean a_ji
13 //
14 // D is a vector of M observed counts (that is, over M bins)
15 // p is a vector of N parameters (that is, over N sources)
16 // A is a vector of vectors of size N x M counts.
17 // f is a vector of vectors of size N x M scale factors.
18 // such that f*A = Aeff, f*dA = sqrt(Aeff), where Aeff
19 // is an effective Poisson count.
20 //
21 // Created: 27-Jun-2005 Harrison B. Prosper
22 // 19-Feb-2011 HBP add add(..)
23 // 16-Apr-2011 HBP allow for parameter fixing
24 // 05-Jun-2016 HBP allow different scaling assuptions for different
25 // sources.
